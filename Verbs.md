# Verbs

A reference of common verbs

## Komen (to come)

| Dutch | English |
| ------- | ----- |
| Ik **kom** | I come |
| Jij **komt** | You come |
| Hij/Zij/Het **komt** | He/She/It comes |
| Wij **komen** | We come |
| Jullie **komen** | You (*pl.*) come |
| Zij **komen** | They come |
