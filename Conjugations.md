# Conjugations

Conjugations (vervoegingen) in Dutch.

All examples use the verb “doen“ (to do)

## Tegenwoordige tijd (now)

| Dutch | English |
| ----- | ------- |
| Ik doe | I do |
| Jij doet | You do |
| Hij/Zij/Het doet | He/She/It does |
| Wij doen | We do |
| Jullie doen | You *pl.* do |
| Zij doen | They do |
