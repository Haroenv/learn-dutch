# Irregular Verbs

## Zijn (to be)

| English | Dutch |
| ------- | ----- |
| I am | Ik **ben** |
| You are | Jij **bent** |
| He/She/It is | Hij/Zij/Het **is** |
| We are | Wij **zijn** |
| You (*pl.*) are | Jullie **zijn** |
| They are | Zij **zijn** |
